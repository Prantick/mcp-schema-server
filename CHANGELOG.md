# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-11-20

### Added
- Initial release of mcp-schema-server
- GenericMcpPlugin class for creating MCP servers from manifest files
- Automatic input validation using JSON Schema (AJV)
- Support for Tools, Resources, and Prompts
- TypeScript support with full type definitions
- Startup integrity checks for handlers
- Automatic error handling and response formatting
- Support for image and text content types
- Comprehensive documentation and examples
- Employee Directory example demonstrating all features

### Features
- Zero-configuration setup
- Schema-driven validation
- Type-safe handler functions
- Flexible return types (JSON, images, etc.)
- Built-in MCP protocol compliance
